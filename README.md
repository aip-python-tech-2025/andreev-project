Проект **посвящён** системе управления *библиотекой*. Она будет ~~очень~~ большой и очень удобной.

Автор проекта: ***Николай Андреев***. [Посмотреть проект в действии.](https://www.youtube.com/watch?v=dQw4w9WgXcQ)

## Возможности бота

1. Добавление книги в `каталог`
2. **Регистрация** читательского билета
3. Выдача книги читателю на срок до 10 дней
   * Продление книги ещё на 10 дней один

Донаты принимаются на:

* Банковскую карту: 4200 0000 0000 0000
* ETH: 0x000000000000000000000000000
* BTC: 0x000000000000000000000000000

Список дел:

* [ ] Пресс
* [x] Турник

### Список команд

| Команда   |                         Описание                         |
|-----------|:--------------------------------------------------------:|
| `/start`  | Запускает бота, регистрирует пользователя, если его нет  |
| `/help`   |              Выводит подсказку по командам               |
| `/borrow` |                  Выдаёт книгу читателю                   |

## Как запустить локально?

Для запуска нужен Python 3.12 и выше.

Пример использования кода:

```python
book = Book('Война и мир', 'Лев Толстой')
print(book)
```
